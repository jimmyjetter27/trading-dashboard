Quick Guide

Enable The Backend

1. Start the Python server

Run `run_metatrader_trade_observer.bat` and keep that terminal window open.

2. Open the local dashboard

Use `http://127.0.0.1:8765` in your browser, not the HTML file directly.

3. Connect to MetaTrader 5

Once the page loads from the local server, click `Connect to MT5`.
